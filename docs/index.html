难度级别

9个等级，渐进式学习路径

Lv.1

Basics 基础

90 个练习

起点。Hello World，变量，print。基础语法，变量类型，输入输出。

  • 数据类型与运算符
  • 控制流与循环
  • 函数与类基础
  • 文件操作与异常处理
Lv.2

Simple 简单

136 个练习

上手。写一些只有十几行的逻辑。条件判断，循环，基础算法逻辑。

  • 基础算法实现
  • 排序与查找
  • 递归与回溯
  • 链表与二叉树
Lv.3

Normal 普通

50 个练习

日常。开始用函数封装，处理文件。函数，文件读写，模块化思维。

  • 函数封装设计
  • 文件读写操作
  • 模块化开发
  • 小型应用开发
Lv.4

Hard 困难

50 个练习

爬坡。引入面向对象（OOP）。类与对象，继承，封装，多态。

  • 类与对象设计
  • 继承与多态
  • 封装与抽象
  • 业务系统开发
Lv.5

Complex 复杂

50 个练习

架构。多模块协作，异常处理。异常捕获，多文件协作，数据库连接。

  • 设计模式实现
  • 多模块协作
  • 异常处理架构
  • 数据库集成
Lv.6

Advanced 进阶

50 个练习

深入。使用外部库，网络请求。requests, pandas, GUI, 数据可视化。

  • 异步编程
  • 依赖注入
  • ORM实现
  • 缓存与路由
Lv.7

Expert 专家

50 个练习

专业。代码规范，完善工具。CLI工具，代码规范，健壮性设计。

  • 元编程技术
  • AST解析
  • 代码生成
  • AOP框架
Lv.8

Master 大师

50 个练习

掌控。从零构建完整系统。Web后端，数据分析工具，系统架构。

  • 事件溯源
  • CQRS架构
  • Saga模式
  • 熔断器设计
Lv.9

Nightmare 噩梦

50 个练习

终极挑战。开发完整分布式系统。Raft共识，分布式KV，MapReduce。

  • Raft共识算法
  • 分布式KV存储
  • 消息队列系统
  • MapReduce框架

如何使用

快速开始你的学习之旅

1

按需学习

根据你的当前水平选择合适的难度级别开始学习。建议从Basics模块开始，循序渐进。

2

先想后看

强烈建议先尝试自己编写，卡住时再查看标准实现，理解后再阅读详细解析加深印象。

3

动手实践

每个练习都是可运行的，建议实际运行代码并尝试修改。同时运行测试用例验证代码正确性。

4

扩展挑战

尝试修改代码实现更多功能，或添加新的测试用例。挑战更高难度级别。
